# E-commerce Analysis - Pet Project

## О проекте

Анализ поведения пользователей интернет-магазина на основе событийных данных. В рамках анализа рассматриваются популярность товаров, поведение в воронке,
ценовая чувствительность и время принятия решения о покупке. Отдельное внимание уделяется проверке применимости классических воронок и поиску альтернативных метрик, лучше описывающих пользовательское поведение.

## Цели анализа

- Исследовать воронку конверсии (просмотр - корзина - покупка)
- Выявить топ категорий и бренды по покупкам
- Проанализировать влияние цены на поведение пользователей
- Рассчитать время принятия решения о покупке

## Технологии

- **Python 3.8+**
- **pandas** - обработка данных
- **SQLite** - SQL анализ

## Как запустить

1. Установите зависимости:
   ```bash
   pip install pandas
   ```

2. Откройте `improved_ecommerce_analysis.ipynb` в Jupyter Notebook

3. Запустите все ячейки (Cell → Run All)

## Основные результаты

- Построена воронка конверсии по категориям
- Выявлено влияние ценовых сегментов на поведение
- Рассчитано время от просмотра до покупки
- Определены категории с высокой конверсией
- Сделаны выводы по работе
